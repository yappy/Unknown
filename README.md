# How to build

## Linux
Only tested in Debian!

### dependencies
* C/C++ toolchains and fundamental tools
  * apt-get build-essential
  * yum groupinstall "Development Tools"
* cmake
* SDL2
  * apt-get install libsdl2-dev
  * yum install SDL2-devel

### build
```
$ cd build/
$ ./build.sh
$ make
```
